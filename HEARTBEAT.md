# Heartbeat Checklist

## Memory Maintenance
**EVERY HEARTBEAT:** Check that dagens memory-fil findes og er opdateret.
- Fil: `memory/YYYY-MM-DD.md`
- Opret hvis den mangler — opdater hvis seneste ændring er > 1 time siden.
- **Vigtigt:** Skriv en kort opsummering af igangværende sager (ingen fluff eller gentagelser). Kun nødvendige fakta, beslutninger og læringer.
- **Langtids-hukommelse:** Hvis der er noget der skal bevares permanent, notér det også i `MEMORY.md` (kun når det er vigtigt).
- Brug **change-only logging**: log kun nye fejl, nye recovery-steps eller nye resultater (spring “alt godt”-noter over).
- Tilføj én kort **EOD-sætning** der opsummerer dagens heartbeat-status.

## System Health
1. Check om alle workspace-agenter er sunde.
2. Hvis noget er rødt, ping Kevin på Telegram med kort status + næste skridt.
3. Hvis alt er grønt, forbliv tavs (ingen rapport).

**Response rules:**
- Svar kun, når noget kræver opmærksomhed, og hold det ultrakort.
- Hvis intet kræver handling, send intet.
