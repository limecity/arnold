# Chris Camillo – Watch & Inspiration

## Formål
- Følge Chris Camillo (Dumb Money / Social Arbitrage) på tværs af offentlige kanaler.
- Notere nye investeringer, forretningsgreb og monetiseringsmetoder.
- Oversætte idéerne til konkrete, hurtige forretningsmuligheder i Danmark.

## Arbejdsgang
1. **Monitorering:** YouTube (Dumb Money), X/Twitter (@ChrisCamillo, @DumbMoneyTV), Instagram, podcast/interviews, evt. Substack/nyhedsbreve.
2. **Log:** For hver observation logges dato, kilde, fokus (trend, produkt, kampagne), estimeret potentiale.
3. **Analyse:** Identificér mønstre (fx retail arbitrage, fællesskabs-baserede medier, niche-subscription).
4. **Forslag:** Udarbejd hurtige DK-konceptidéer (beskrivelse, hvordan vi tjener penge, krav, risiko, næste skridt).
